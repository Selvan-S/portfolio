# **App Name**: TechTask TypeScript, ReactJS, Tailwind

## Core Features:

- Personal Overview: A prominent section displaying Selvan's name, title, a brief professional summary, and key contact information.
- Skillset Display: A well-organized and visually appealing section to showcase Selvan's technical proficiencies, categorized for clarity.
- Experience Timeline: A chronological and engaging presentation of Selvan's professional experience, highlighting key responsibilities and achievements.
- Project Showcase: A dedicated portfolio section for personal projects, featuring descriptions, tech stacks, and links to GitHub repositories or live demos.
- AI Content Refiner Tool: An internal tool to assist Selvan in generating or enhancing concise and impactful descriptions for his projects or professional summary, utilizing a generative AI model.
- Interactive Contact Form: A simple form allowing visitors to send messages directly, facilitating easy communication for inquiries or collaborations.

## Style Guidelines:

- Background color: A very dark, subtle blue-purple (#1D1B24) providing a sleek, tech-oriented foundation.
- Primary accent color: A modern lavender-purple (#7E72FF) to highlight key elements and convey creativity and technological focus against the dark background.
- Secondary accent color: A professional mid-tone blue (#4D89E0) to complement the primary color and ensure clear hierarchy and visual interest.
- Headline and body text font: 'Inter' (grotesque-style sans-serif), chosen for its modern, clean appearance and excellent readability across various content types.
- Code snippets font: 'Source Code Pro' (monospaced sans-serif), ensuring legibility and professional presentation for any displayed code.
- Utilize a consistent set of minimalist line-art icons for navigation, skill indicators, and social media links, aligning with the clean aesthetic.
- Adopt a clean, spacious, and grid-based layout, emphasizing content through ample negative space and clear visual hierarchy for a minimalistic feel.
- Implement subtle, elegant micro-interactions and transitions (e.g., hover effects, content fades) to enhance the user experience without distracting from the information.
